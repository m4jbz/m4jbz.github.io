Software que uso y recomiendo

  • dwm. Gestor de ventanas del proyecto suckless, muy customisable y liviano.
  • Archlinux. La mejor distro de GNU/Linux, nada mas que decir.
  • Librewolf. Navegador seguro y que respeta la privacidad del usuario.
  • Neovim. El mejor editor de texto, liviano, customisable y bonito.
  • LaTeX. Sistema de composición de documentos pdf, muy bueno para hacer documentos o presentaciones elegantes.
  • st. Como terminal uso la suckless terminal con el fork de LukeSmith.
  • ncmpcpp. Cliente de mpd el cual es un daemon o servicio de música.
  • lf. Explorador de archivos muy ligero y customisable.
  • dmenu. Programa que permite ejecutar otros programas del sistema, así como mostrar texto en la parte superior de la pantalla, sirve mucho para scripts en shell.
  • Mangareader. Programa para leer mangas es GNU/Linux.
  • qbittorrent. Cliente para descargar archivos torrent o links magnet.
